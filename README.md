## Project Details

### Title: *Course Enrollment Management System*

**Description:** Create a system to track enrollment inuniversity courses.

### Requirements

Create a base class Enrollment with attributes:
- `enrollID`
- `courseCode`
- `studentName`
- `semester`

Create subclasses with additional attribute:
- UndergraduateEnrollment:
    - `programme`
    - `studyYear`
- PostgraduateEnrollment:
    - `researchArea`
    - `supervisorName`

Array of Objects:
- Use an array to store Enrollment objects.

### Functionalities

1. Add, view, search, update, and delete enrollment records.
2. Search by course code or student name.
3. Filter by semester or level (Undergraduate/ Postgraduate).

### Additional Features

1. Generate course enrollment statistics.
2. Validate course code format.
